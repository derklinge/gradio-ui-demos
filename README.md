## Installation

### Prerequisites

- Python 3.8+

### Clone the repository

```sh
git clone https://github.com/derklinge/gradio-ui-demos
```

### Install gradio package

Gradio is required to build and run this repository. Install using PIP:

```sh
pip install gradio
```

## First run

```sh
python3 gradio-ui-demos
```
